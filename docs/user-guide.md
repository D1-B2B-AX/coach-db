## 1. 개요

### 1.1 시스템 소개

Coach DB는 실습코치 정보를 한 곳에서 조회·탐색·연락하는 코치 통합 관리 도구입니다.

### 1.2 주요 기능

| 기능 | 설명 |
|------|------|
| 코치 스케줄 조회 | 월별 캘린더에서 코치 가용 현황 확인 |
| 코치 정보 관리 | 코치 등록, 수정, 삭제, 상세 조회 |
| 투입 이력 관리 | 코치별 과정 투입 이력 등록 및 조회 |
| 스케줄 입력 요청 | 코치에게 가용 스케줄 입력 링크 전달 |
| 엑셀 추출 | 코치 정보를 엑셀 파일로 다운로드 |
| 관리자 기능 | 매니저 권한 관리, 삭제 코치 복원, 링크 관리, 메일 발송 |

### 1.3 접속 환경

- **브라우저**: Chrome, Edge, Safari 등 최신 브라우저 권장
- **계정**: `@day1company.co.kr` Google 계정 필요

### 1.4 화면 구성

```
┌──────────────────────────────────────────────┐
│  Coach DB        대시보드  코치  [관리자]  로그아웃  │  ← 상단 내비게이션
├──────────────────────────────────────────────┤
│                                              │
│                 본문 영역                      │
│                                              │
└──────────────────────────────────────────────┘
```

- **대시보드**: 월별 코치 가용 현황
- **코치**: 코치 목록 조회 및 관리
- **관리자**: 시스템 관리 (관리자 권한 계정에만 표시)

## 3. 대시보드

### 3.1 화면 설명

이번 달 코치들의 가용 스케줄을 한눈에 확인하는 메인 화면입니다.

### 3.2 화면 구성

```
┌──────────────────────────────────────────────┐
│  ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│  │ 입력완료  │ │  접속만   │ │  미확인   │     │  ← (A) 현황 카드
│  │   12명   │ │   5명    │ │   8명    │     │
│  └──────────┘ └──────────┘ └──────────┘     │
│                                              │
│  ◀  2026년 3월  ▶            [새로고침]       │  ← (B) 월 탐색
│  ┌─────────────────────────┐                 │
│  │ 일  월  화  수  목  금  토 │                │  ← (C) 캘린더
│  │  1   2   3   4   5   6   7│                │
│  │  ...                      │                │
│  └─────────────────────────┘                 │
│                                              │
│  ┌─ 시간 ──┐ ┌─ 분야 ──┐ ┌─ 평가 ──┐       │  ← (D) 필터
│                                              │
│  ┌──────────────────────────────────────┐    │
│  │ 코치명   가용시간   분야   평점  최근투입 │   │  ← (E) 코치 목록
│  │ ...                                   │    │
│  └──────────────────────────────────────┘    │
└──────────────────────────────────────────────┘
```

### 3.3 영역별 설명

**(A) 현황 카드**

해당 월의 코치 스케줄 입력 현황을 요약합니다.

| 카드 | 색상 | 의미 |
|------|------|------|
| 입력완료 | 초록 | 코치가 가용 스케줄을 입력 완료 |
| 접속만 | 주황 | 코치가 링크에 접속했지만 저장하지 않음 |
| 미확인 | 빨강 | 코치가 아직 접속하지 않음 |

**(B) 월 탐색**

- `◀` `▶`: 이전/다음 달 이동
- `새로고침`: 최신 데이터 수동 갱신 (자동 30초 갱신)

**(C) 캘린더**

- 날짜 클릭 → 해당 날짜에 가용한 코치 목록이 하단에 표시

**(D) 필터**

| 필터 | 옵션 | 설명 |
|------|------|------|
| 시간대 | 전체 / 09~12 / 12~18 / 18~22 | 원하는 시간대의 코치만 표시 |
| 분야 | 체크박스 복수 선택 | 선택한 분야의 코치만 표시 |
| 평가 | 전체 / 4점 이상 / 3점 이상 / 신규 | 평점 기준 필터 |

**(E) 코치 목록**

선택한 날짜 + 필터 조건에 맞는 코치가 카드 형태로 표시됩니다.

| 표시 항목 | 내용 |
|-----------|------|
| 코치명 | 이름 |
| 가용 시간 | 예: 09:00~12:00, 14:00~18:00 |
| 분야 | 전문 분야 태그 |
| 평점 | ★ 평균 점수 |
| 최근 투입 | 마지막 투입 과정명 및 종료 시기 |

### 3.4 주요 조작 절차

**특정 날짜의 가용 코치를 확인하려면:**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 캘린더에서 원하는 날짜 클릭 | 해당 날짜가 선택되고 코치 목록 표시 |
| 2 | (선택) 필터 조건 설정 | 조건에 맞는 코치만 표시 |
| 3 | 코치 카드 확인 | 이름, 시간, 분야, 평점 확인 |

## 4. 코치 목록

### 4.1 화면 설명

등록된 전체 코치를 조회하고 검색·필터·엑셀 추출하는 화면입니다.

### 4.2 화면 구성

```
┌──────────────────────────────────────────────┐
│  코치 목록                         [+ 등록]   │
│                                              │
│  ┌─ 검색... ─────────┐  ┌─ 분야: 전체 ──┐   │
│                                              │
│  ☐ 전체 선택    총 272명   선택: 0명  [엑셀 추출]│
│  ┌──────────────────────────────────────┐    │
│  │ ☐  ● 김코치   010-1234-5678  React  ★4.2│ │
│  │ ☐    박코치   010-2345-6789  Java   ★3.8│ │
│  │ ☐  ● 이코치   010-3456-7890  PM     -   │ │
│  │ ...                                   │    │
│  └──────────────────────────────────────┘    │
└──────────────────────────────────────────────┘
```

> **참고**: 이름 왼쪽의 ● 초록색 점은 "활동중" 상태를 의미합니다.

### 4.3 검색 및 필터

| 기능 | 조작 | 설명 |
|------|------|------|
| 검색 | 검색창에 이름 입력 | 입력 즉시 자동 필터링 |
| 분야 필터 | 드롭다운에서 분야 선택 | 복수 선택 가능, `초기화`로 리셋 |

### 4.4 엑셀 추출 절차

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 추출할 코치의 체크박스 선택 (또는 `전체 선택`) | 선택 인원수 표시 |
| 2 | `엑셀 추출` 클릭 | .xlsx 파일 다운로드 |

### 4.5 코치 상세 이동

- 코치 행을 클릭하면 해당 코치의 상세 페이지로 이동합니다.

## 5. 코치 등록

### 5.1 화면 설명

새로운 코치를 시스템에 등록하는 화면입니다.

### 5.2 진입 경로

`코치 목록` → `+ 등록` 버튼 클릭

### 5.3 입력 항목

| 항목 | 필수 | 입력 방식 | 설명 |
|------|:----:|-----------|------|
| 이름 | ✅ | 텍스트 입력 | 코치 이름 |
| 생년월일 | | 날짜 선택 | |
| 전화번호 | | 텍스트 입력 | 예: 010-0000-0000 |
| 이메일 | | 텍스트 입력 | |
| 소속 | | 텍스트 입력 | 소속 기관/회사 |
| 근무유형 | | 텍스트 입력 | 예: 실습코치, 운영조교 |
| 시급 | | 숫자 입력 | 원(₩) 단위 |
| 가능 분야 | | 검색+선택 | 복수 선택 가능, 직접 추가 가능 |
| 가능 커리큘럼 | | 검색+선택 | 복수 선택 가능, 직접 추가 가능 |
| 코치 메모 | | 텍스트 입력 | 코치 관련 참고 메모 |
| 내부 메모 | | 텍스트 입력 | 매니저 전용 내부 메모 |

### 5.4 등록 절차

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 코치 목록에서 `+ 등록` 클릭 | 등록 폼 표시 |
| 2 | 필수 항목(이름) 포함 정보 입력 | |
| 3 | `저장` 클릭 | 등록 완료 팝업 표시 |
| 4 | 팝업에서 스케줄 입력 링크 `복사` | 클립보드에 링크 복사 |
| 5 | 복사한 링크를 코치에게 전달 | 코치가 링크로 스케줄 입력 가능 |

> **참고**: 등록 완료 후 표시되는 스케줄 입력 링크는 해당 코치 전용입니다. 이 링크를 통해 코치가 직접 가용 스케줄을 입력할 수 있습니다.

## 6. 코치 상세 조회

### 6.1 화면 설명

개별 코치의 프로필, 스케줄, 투입 이력을 확인하는 화면입니다. 3개 탭으로 구성됩니다.

### 6.2 진입 경로

`코치 목록` → 코치 행 클릭

### 6.3 상단 정보

| 표시 항목 | 설명 |
|-----------|------|
| 코치 이름 | |
| 상태 배지 | `활동중`(초록) / `비활동`(회색) / `휴직`(주황) |
| 평균 평점 | ★ 점수 |

### 6.4 탭 1: 프로필

코치의 기본 정보, 포트폴리오, 분야/커리큘럼, 메모를 확인합니다.

**기본 정보**

| 항목 | 내용 |
|------|------|
| 생년월일 | |
| 소속 | 소속 기관/회사 |
| 근무유형 · 시급 | 예: 실습코치 · ₩30,000 |
| 연락처 | 전화번호 |
| 이메일 | |

**포트폴리오**

| 조작 | 결과 |
|------|------|
| `+ 포트폴리오 추가` 클릭 또는 파일 드래그 | 파일 업로드 |
| 파일명 클릭 | 파일 다운로드 |
| `×` 클릭 | 파일 삭제 |

**분야 & 커리큘럼**

- 가능 분야: 파란색 태그
- 가능 커리큘럼: 회색 태그

**메모**

| 구분 | 설명 | 수정 |
|------|------|------|
| 코치가 남긴 메모 | 코치 관련 참고 메모 | `수정` → 내용 변경 → `저장` |
| 내부 메모 | 매니저 전용 메모 (코치에게 비공개) | `수정` → 내용 변경 → `저장` |

**수정 이력**

- 하단 `수정 이력` 클릭 시 펼침
- 변경자, 변경 일시, 변경 항목(이전값 → 변경값) 표시

### 6.5 탭 2: 스케줄

코치가 입력한 월별 가용 스케줄을 확인합니다.

**캘린더 색상**

| 색상 | 의미 |
|------|------|
| 연두색 배경 | 가용 — 코치가 가능하다고 입력한 날 |
| 파란색 배경 | 확정 — 투입이 확정된 날 |

**접속 상태 배지** (캘린더 상단 표시)

| 배지 | 의미 |
|------|------|
| 미확인 | 코치가 아직 링크에 접속하지 않음 |
| 접속만 | 접속은 했지만 스케줄을 저장하지 않음 |
| 입력완료 | 스케줄 입력 완료 |

**날짜 상세 확인**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 캘린더에서 날짜 클릭 | 해당 날짜가 주황색으로 선택 표시 |
| 2 | 오른쪽 패널 확인 | 상세 가용 시간대 및 확정 투입 정보 표시 |

### 6.6 탭 3: 투입 이력

코치의 과정 투입 이력을 조회하고 관리합니다.

**이력 카드 표시 정보**

| 항목 | 내용 |
|------|------|
| 상태 | `예정`(파랑) / `진행중`(주황) / `완료`(회색) / `취소`(빨강) |
| 과정명 | 투입된 교육 과정명 |
| 기간 | 시작일 ~ 종료일 |
| 시간 · 장소 | 투입 시간대 및 장소 |
| 담당 매니저 | 해당 투입을 등록한 매니저 |
| 평점 · 재섭외 | ★ 평점, 재섭외 희망/비희망 |
| 피드백 | 투입 후 피드백 |

**이력 등록 절차**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | `+ 이력 등록` 클릭 | 등록 팝업 표시 |
| 2 | 항목 입력 (아래 표 참고) | |
| 3 | `저장` 클릭 | 이력 목록에 추가 |

**이력 등록 입력 항목**

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 코스명 | ✅ | 교육 과정명 |
| 시작일 | ✅ | 투입 시작일 |
| 종료일 | ✅ | 투입 종료일 |
| 시작/종료 시간 | | 투입 시간대 |
| 장소 | | 투입 장소 |
| 상태 | | 예정 / 진행중 / 완료 / 취소 |
| 평점 (1~5) | | 코치 평점 (미평가 선택 가능) |
| 피드백 | | 텍스트 입력 |
| 재고용 의사 | | 체크박스 |
| 담당자 | | 담당 매니저명 |

> **참고**: 투입 이력은 본인이 등록한 건만 수정할 수 있습니다. 관리자(admin)는 모든 이력을 수정할 수 있습니다.

## 9. 관리자 기능

> **접근 권한**: 관리자(admin) 역할이 부여된 계정만 사용할 수 있습니다. 상단 메뉴에 `관리자`가 표시되지 않는다면 관리자에게 권한을 요청하세요.

### 9.1 진입 경로

상단 메뉴 → `관리자` 클릭

### 9.2 탭 1: 매니저 관리

등록된 매니저의 역할을 관리합니다.

**역할 종류**

| 역할 | 색상 | 권한 |
|------|------|------|
| 관리자 | 초록 배지 | 모든 기능 + 관리자 메뉴 접근 |
| 일반 | 파랑 배지 | 기본 매니저 기능 (코치 조회/등록/수정/삭제, 투입 이력 관리) |
| 차단 | 빨강 배지 | 로그인 차단 |

**개별 역할 변경**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 매니저 목록에서 대상 확인 | |
| 2 | 역할 드롭다운 클릭 | 관리자 / 일반 / 차단 선택 |
| 3 | 원하는 역할 선택 | 즉시 변경 적용 |

**일괄 역할 변경**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 대상 매니저 체크박스 선택 (또는 전체 선택) | 선택 인원수 표시 |
| 2 | `일괄 변경` 드롭다운에서 역할 선택 | 선택된 전원의 역할 일괄 변경 |

### 9.3 탭 2: 삭제된 코치

삭제된 코치를 복원하거나 영구 삭제합니다.

**복원 절차**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | 삭제된 코치 목록에서 대상 확인 | 삭제자·삭제일 확인 가능 |
| 2 | `복원` 클릭 | 확인 팝업 표시 |
| 3 | 확인 | 코치가 목록에 복원 |

**완전 삭제 절차**

| 단계 | 조작 | 결과 |
|------|------|------|
| 1 | `완전 삭제` 클릭 | 확인 팝업 표시 |
| 2 | 확인 | 영구 삭제 완료 |

> **주의**: `완전 삭제`는 되돌릴 수 없습니다. 해당 코치의 모든 데이터가 영구적으로 삭제됩니다.

### 9.4 탭 3: 코치 링크

코치별 스케줄 입력 링크를 조회하고 관리합니다.

| 기능 | 조작 | 설명 |
|------|------|------|
| 검색 | 이름, 연락처, 이메일 입력 | 목록 필터링 |
| 복사 | `복사` 클릭 | 해당 코치의 스케줄 입력 링크가 클립보드에 복사 |
| 재발급 | `재발급` 클릭 → 확인 | 새 링크 생성 |

> **주의**: 링크를 재발급하면 기존 링크는 즉시 무효화됩니다. 코치에게 새 링크를 반드시 재전달해야 합니다.

### 9.5 탭 4: 입력 메일 발송

코치들에게 스케줄 입력을 요청하는 메일 발송을 준비합니다.

**화면 구성**

```
┌──────────────────────────────────────────────┐
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌──────┐  │
│  │입력완료 │ │ 접속만  │ │ 미확인  │ │ 전체  │  │  ← (A) 현황 카드
│  │  12명  │ │  5명   │ │  8명   │ │ 25명  │  │
│  └────────┘ └────────┘ └────────┘ └──────┘  │
│                                              │
│  ☐ 전체 선택                     [이메일 복사] │  ← (B) 코치 선택
│  ☐ 김코치  coach1@email.com                   │
│  ☐ 박코치  coach2@email.com                   │

│  ─── 메일 문구 ──────────────── [문구 복사] ── │  ← (C) 메일 템플릿

│  본문: ...                                    │

```

**메일 발송 절차**

| 단계 | 조작 | 결과 |
|------|------|------|

> **참고**: 기본 메일 템플릿이 제공됩니다. 제목과 본문을 자유롭게 수정할 수 있습니다.

## 10. 코치 스케줄 입력 안내

> 이 섹션은 코치에게 전달할 안내 내용입니다. 매니저가 직접 사용하는 화면이 아닙니다.

### 10.1 링크 형태

```
https://coach-db.up.railway.app/coach?token=고유토큰
```

- 코치 등록 시 자동 생성됩니다.
- 관리자 메뉴 > 코치 링크 탭에서 확인/복사/재발급할 수 있습니다.
- 각 코치마다 고유한 링크이며, 별도 로그인 없이 접속 가능합니다.

### 10.2 코치 화면 안내

코치가 링크에 접속하면 다음 기능을 사용할 수 있습니다:

| 기능 | 설명 |
|------|------|
| 월간 캘린더 | 날짜를 선택하여 가용일 표시 |
| 시간대 선택 | 30분 단위로 가용 시간 입력 |
| 확정 투입 확인 | 이미 확정된 투입 일정이 파란색으로 표시 (수정 불가) |
| 투입 이력 확인 | 과거 투입 이력 조회 |
| 저장 | 입력 완료 후 반드시 `저장` 클릭 필요 |

> **참고**: 코치에게 링크 전달 시 "저장 버튼을 꼭 눌러주세요"를 안내해 주세요. 저장하지 않으면 `접속만` 상태로 남습니다.

## 12. FAQ

**Q. 로그인이 되지 않습니다.**
> @day1company.co.kr 계정으로 로그인해야 합니다. 외부 계정은 접근이 제한됩니다. 문제가 지속되면 관리자에게 계정이 `차단` 상태인지 확인을 요청하세요.

**Q. 상단 메뉴에 `관리자`가 보이지 않습니다.**
> 관리자(admin) 권한이 있는 계정에만 표시됩니다. 권한이 필요하면 기존 관리자에게 역할 변경을 요청하세요.

**Q. 코치의 스케줄이 `미확인`으로 계속 남아있습니다.**
> 코치가 아직 링크에 접속하지 않은 상태입니다. 관리자 > 입력 메일 발송 탭에서 해당 코치에게 입력 요청 메일을 발송하세요.

**Q. 코치 링크를 분실했습니다.**
> 관리자 > 코치 링크 탭에서 해당 코치의 링크를 `복사`할 수 있습니다. 링크가 작동하지 않으면 `재발급`하세요.

**Q. 삭제한 코치를 복구하고 싶습니다.**
> 관리자 > 삭제된 코치 탭에서 `복원` 버튼을 클릭하세요. 단, `완전 삭제`된 코치는 복구할 수 없습니다.

**Q. 다른 매니저가 등록한 투입 이력을 수정할 수 없습니다.**
> 투입 이력은 등록한 본인만 수정 가능합니다. 관리자(admin)에게 수정을 요청하세요.

**Q. 엑셀 추출 시 어떤 정보가 포함되나요?**
> 선택한 코치의 이름, 연락처, 이메일, 소속, 분야, 평점 등 기본 정보가 포함됩니다.
